# AI Explanation Policy

이 문서는 AI 분석 브랜치가 따라야 하는 해설 품질 기준이다. Foundation 브랜치에서는 타입과 문서만 준비하며 실제 OpenAI API 호출은 구현하지 않는다.

## 1. 해설의 목적

기사를 짧게 줄이는 것이 목적이 아니다.

경제를 모르는 사람이 다음 인과관계를 이해하게 해야 한다.

```text
기존 상황
→ 변화
→ 변화 이유
→ 직접 영향
→ 간접 영향
→ 앞으로 가능한 흐름
→ 사용자가 확인할 사항
```

## 2. 문체

- 존댓말을 사용한다.
- 한 문장을 짧게 작성한다.
- 전문용어를 그대로 두지 않는다.
- 어려운 표현 뒤에는 쉬운 설명을 붙인다.
- 과도하게 공포를 조성하지 않는다.
- 투자 수익을 약속하지 않는다.
- 사용자를 무시하거나 지나치게 어린아이처럼 대하지 않는다.

## 3. 좋은 설명의 예

기사 내용:

> 금융당국이 전세보증금 별도 관리 방안을 검토하고 있다.

좋은 설명:

```text
기존에는 집주인이 세입자에게 받은 전세보증금을 직접 보유했습니다.
집주인은 이 돈을 다른 부동산을 사거나 대출을 갚는 데 사용할 수도 있었습니다.

새 제도가 도입되면 전세보증금을 별도 기관이나 계좌에서 관리할 가능성이 있습니다.
세입자 입장에서는 보증금을 돌려받지 못할 위험이 줄어들 수 있습니다.

반면 집주인은 보증금을 자유롭게 활용하기 어려워질 수 있습니다.
그 결과 일부 집주인이 전세 대신 월세를 선택할 가능성도 있습니다.

다만 현재는 검토 단계이므로 실제 제도 내용과 시행 시점은 확정되지 않았습니다.
```

## 4. 나쁜 설명의 예

```text
정부가 전세금을 관리합니다.
세입자에게 좋은 정책입니다.
전세가 사라질 것입니다.
```

문제점:

- 기존 상황이 없다.
- 인과관계가 없다.
- 확정되지 않은 결과를 단정한다.
- 사용자 생활에 어떤 변화가 있는지 부족하다.

## 5. 확정 사실과 예상의 구분

### 확정 사실

```text
한국은행이 기준금리를 0.25%포인트 내렸습니다.
```

### 검토 또는 제안

```text
정부가 전세보증금 관리 방안을 검토하고 있습니다.
```

### 예상

```text
은행의 예금금리와 대출금리가 내려갈 가능성이 있습니다.
```

예상 문장에는 다음 표현 중 하나를 포함한다.

- 가능성이 있습니다.
- 영향을 줄 수 있습니다.
- 이어질 수 있습니다.
- 예상됩니다.
- 아직 확정된 것은 아닙니다.

## 6. 경제용어 설명 기준

나쁜 설명:

```text
DSR은 총부채원리금상환비율입니다.
```

좋은 설명:

```text
DSR은 연소득과 비교했을 때 1년 동안 갚아야 하는 모든 대출의 원금과 이자가 얼마나 되는지 보는 기준입니다.
은행은 이 비율을 이용해 추가 대출을 갚을 능력이 있는지 판단합니다.
```

가능하면 생활 예시를 추가한다.

```text
연봉이 5,000만 원이고 1년에 갚는 대출 원금과 이자가 2,000만 원이면 DSR은 약 40%입니다.
```

## 7. 행동 안내 기준

권장:

- 대출 예정자는 은행별 실제 금리를 확인하세요.
- 전세계약 전 보증보험 가입 가능 여부를 확인하세요.
- 정책이 확정되기 전 계약 조건을 단정하지 마세요.
- 해당 지원의 소득·자산 조건을 확인하세요.

금지:

- 지금 당장 이 주식을 사세요.
- 집값이 무조건 오르니 지금 매수하세요.
- 이 상품은 반드시 수익이 납니다.
- 대출을 최대한 받아 투자하세요.

## 8. 여러 출처가 있는 경우

동일 사건을 여러 언론사가 보도하면 하나의 뉴스로 묶는다.

가능한 우선순위:

1. 정부·공공기관 원문
2. 중앙은행·금융당국
3. 공식 통계기관
4. 주요 언론사
5. 기타 출처

언론사별 해석이 다르면 하나의 해석만 정답처럼 제시하지 않는다.

## 9. 정보가 부족한 경우

기사가 짧거나 근거가 부족하면 내용을 만들어내지 않는다.

사용 가능한 표현:

- 기사만으로는 구체적인 시행 방식이 확인되지 않습니다.
- 적용 대상은 추가 발표를 확인해야 합니다.
- 실제 금리 변화 폭은 은행마다 다를 수 있습니다.
- 현재 자료만으로 가격 상승 여부를 단정하기 어렵습니다.

## 10. 개인화 기준

기본 사용자는 다음 상황을 가진다.

- 경제 초보자
- 신혼부부
- 주택 마련에 관심
- 출산 준비 중
- 예금, 대출, 전세, 생활비에 관심

개인화는 해당 상황과 직접 관련 있는 경우에만 사용한다.

억지로 모든 기사에 신혼부부 영향을 붙이지 않는다.
