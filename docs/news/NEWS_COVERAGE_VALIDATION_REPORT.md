# RSS 수집 커버리지 검증 보고서

## 1. 검증 목적

뉴스 수집 소스(7개)와 분류·관련성·다양성 로직 개선 후, 실제 RSS 데이터를 이용하여:

- 경제·재테크·주식·부동산 정보가 충분히 수집되는지 확인
- 최종 브리핑이 특정 언론사나 기업 뉴스에 편중되지 않는지 검증
- 필요한 최소한의 마지막 튜닝 수행

## 2. 실행 일시와 시간 범위

- 실행 일시: 2026-07-18 20:09 KST
- 수집 범위: 최근 24시간 (2026-07-17 20:09 ~ 2026-07-18 20:09)
- 운영 스케줄: 1시간 단위 (COLLECTION_WINDOW_MS = 1시간)
- 비고: 24시간 범위는 측정 스크립트 전용. 운영 로직의 수집 범위는 변경하지 않음.

## 14. 발견된 문제와 수정 내용

### 문제 1: 한국경제·서울경제 품질 검증 실패

- **원인**: RSS 피드가 제목·URL·날짜만 제공하고 description/content 필드가 비어 있음
- **영향**: 12건 기사 전량 손실 (전체 수집량의 18%)
- **수정**: `qualityValidator.ts`에서 요약·내용 필수 조건 제거. 제목·URL·날짜만으로 통과 가능하도록 변경
- **위험**: 제목만 있는 저품질 기사가 통과할 수 있으나, 카테고리 분류와 관련성 점수에서 걸러짐

### 문제 2: 관련성 키워드 매칭 범위 협소

- **원인**: "코스피", "코스닥", "증시", "유가", "기름값" 등 개인 재테크에 직접 관련 있는 키워드가 2점으로 설정됨. 최소 관련성 기준 3점에 미달하여 전량 제외
- **영향**: 51건 중 43건(84%)이 기본 2점만 받아 다양성 선택 후보가 7건에 불과
- **수정**: 개인 재테크에 직접적인 키워드를 2점에서 3점으로 상향
  - 이동: "코스피", "코스닥", "증시", "기름값"
  - 추가: "국제유가", "유가 상승", "유가 하락", "분양", "집값", "연금보험", "노후 대비", "노후 준비", "노후 생활", "노후 자금", "노후 불안"

### 문제 3: 키워드 부분 문자열 거짓양성

- **원인**: "전세" 키워드가 "전세계"(worldwide)의 부분 문자열로 매칭. "유가" 키워드가 "유가족"(bereaved family)의 부분 문자열로 매칭. "노후" 키워드가 "노후 석탄화력"(aging coal plant)의 문맥에서 매칭.
- **영향**: 원전 SMR 기사가 전월세·연금 카테고리로 분류됨. 교사 추모 기사가 물가·소비 카테고리로 분류됨.
- **수정**:
  - `categoryClassifier.ts`: "전세" → "전세 "(공백 포함) 및 구체적 복합어("전세가", "전세금" 등)로 교체
  - `categoryClassifier.ts`: "유가" → "국제유가", "유가 상승", "유가 하락"으로 교체
  - `categoryClassifier.ts`: "노후" → "노후 대비", "노후 준비", "노후 생활" 등 구체적 문구로 교체
  - `relevanceScorer.ts`: 동일한 원칙 적용

## 18. 추가 소스 필요 여부

**현재 7개 소스로 운영 가능.**

- 주식·ETF 전용 기사: 현재 소스에서 시장 동향은 수집되나, 종목 분석·ETF 추천 등은 부족. 증권 전문 RSS(예: 한국경제 증권 카테고리, 이데일리 증권) 추가 시 보완 가능
- 금융상품 전용 기사: ISA·CMA·IRP 등 상품 비교 기사는 현재 소스에서 거의 수집 안 됨. 금융감독원 보도자료 RSS 또는 금융 비교 플랫폼 RSS 고려 가능
- 한국경제·서울경제: RSS에 요약이 제공되면 품질 크게 개선됨. 다른 카테고리 RSS(부동산, 증권) 추가 검토 가능

**즉시 추가하지 않음. 모니터링 후 판단.**

## 19. 남은 한계

1. **한국경제·서울경제 RSS 제목만 제공**: 카테고리 분류와 관련성 점수 모두 제목에만 의존하여 정확도 낮음. 특히 제목에 관련 키워드가 없는 기사는 분류·채점 모두 실패
2. **키워드 부분 문자열 한계**: 한국어 특성상 복합어 내 키워드 매칭 거짓양성 완전 제거 불가. "건설"(housing 키워드)이 원전 건설에 매칭되는 등의 사례 잔존
3. **동일 사건 교차 소스 중복**: 요약이 없는 소스의 기사는 요약 기반 유사도 비교가 불가하여 중복 탐지율 저하
4. **토요일/일요일 기사 감소**: 주말에는 경제·재테크 기사 생산량 자체가 적어 카테고리 커버리지 저하 불가피
5. **1시간 수집 윈도우**: 운영 환경에서 1시간 단위로 수집하므로 24시간 측정보다 기사량이 현저히 적음
